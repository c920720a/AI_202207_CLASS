### 인코딩
1. 라벨인코딩
  : 변수형 값을 숫자로 변경
  
2. 원핫인코딩
  - 0, 1를 갖는 벡터로 구성된다.
  - 범주형 값의 개수만큼 feature가 형성된다.
  - 텍스트 데이터를 변경하는 방법론 중 하나.
  - 희소행렬
  - LabelEncoder(), OneHotEncoder()
  - pd.get_dummies()
  
새로운 특성(feature) 선택하기
  - 일변량 통계(통계의 f-value를 이용) target값과 유의한 변수를 선택
  - 모델 기반 선택. 모델에서 중요하다고 생각하는 변수를 선택.
  - 재귀적 모델 기반 선택. 하나의 변수를 선택할 때마다 다른 모델을 이용.
  
교차 검증
  - 데이터가 적을 때 좀 더 많은 데이터를 이용해서
    학습을 하고자 할 때 시작한다.
  - 좀 더 일반화된 모델을 만들고자 할 때 사용될 수 있다.
