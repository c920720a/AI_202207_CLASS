# 지도학습
  - 회귀(Regression)
    예측하고자 하는 target 값이 연속형
  - 분류(Classification)
    예측하고자 하는 target 값이 범주형
	- 이진분류(이항분류) : 범주형의 범주의 개수가 2개
	- 다항분류 : 범주형의 범주의 개수가 3개 이상

# 과적합 or 과대적합 (overfitting)
 - 학습한 것에만 딱 맞게 학습하는 것
 - 학습 데이터에만 성능이 최적회되어서 일반적인 다른 데이터는 잘 최적화되어 있지 않다.
 
# 과소적합(Underfitting)
 - 과대적합의 현상과 반대
 
# K-NN 모델
 - [분류] 새로운 데이터와 기존 데이터 거리 가까운 것
   k=3 이면 세 개의 데이터의 target의 값이 많은 값으로 분류.
   
 - [회귀] 새로운 데이터와의 거리에 따라 k가 3이라고 하면
   세 개의 값의 target의 값 평균으로 예측하는 모델
   
# 선형회귀(Linear Regression)
 - 선형 함수로 표현이 되어 있다.
 - 특징 하나 - 직선, 특징이 두 개 - 평면, 특징이 세 개 - 초평면
 - 선형 모델에서 MSE가 최소가 되는 w, b가 정해진다.
   w : 회귀계수(가중치), b :편향
