 - K-최근접 이웃
    - 하이퍼 파라미터 k 
	- 새로운 데이터가 들어왔을때, k개의 이웃의 값을 보고 판단.
	   회귀 : k개의 이웃이 예측하는 target의 평균
	   분류 : k개의 이웃이 예측하는 target이 많은 다수의 값의 예측
    - 메모리 자원 소모가 큰 편 
    - 많은 데이터 양에는 적당하지 않음.
 
 - 선형회귀 
    - y = w1*x1 + w2*x2 + .... + b
	w1, w2, b : 모델 파라미터 (모델이 결정해 주는 파라미터)
	- MSE(평균 제곱 오차)를 최소화하는 파라미터를 찾는다.
	- 과대적합을 해소 시켜 주는 방법
	   - 릿지 회귀, 랏소 회귀, 엘라스틱넷
	
 - 데이터 전처리 
    표준화 - 평균을 0, 분산을 1로 변경 
	정규화 - 모든 특성이 0과 1사이에 위치하도록 데이터를 변경 
	
    원핫인코딩
	  값을 0,1을 갖는 벡터 형태로 만들어준다.
	  
	구간분할 
	  수치형 데이터를 구간별 대표값으로 만들어준다.
	 
	비선형 변환
	  log - 값의 분포를 정규화를 시켜준다.
	  
	  
 - 평가지표 
    회귀
	  MAE  : 오차의 절대값의 평균 
	  MSE  : 평균 제곱 오차 
	  RMSE : 평균 제곱근 오차 
	  RMLSE : 예측값, 실제값 -> 1을 더해서 log처리.
	  MAPE  
	  
	분류 
	  혼동행렬(confusion matrix) or 오차행렬
	    - 정확도
		- 정밀도(precision)
		- 민감도(Sensitivity, Recall, Tprate)
		- 특이도
		- Fprate
		- ROC Curve (y축이 민감도, x축이 1-특이도)
		- AUC
		- f-score (정밀도, 민감도)
